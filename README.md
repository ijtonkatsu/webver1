# Tonkatsu Injeong Restaurant Website

A premium restaurant website for Tonkatsu Injeong, featuring authentic Japanese tonkatsu cuisine.

## Features

- Responsive design for all devices
- Modern UI with Bootstrap 5
- Multiple pages: Home, About, Menu, Story, Gallery, Blog, Contact
- Custom restaurant theme styling
- PHP-based dynamic content

## Technology Stack

- PHP
- HTML5
- CSS3 (Bootstrap 5)
- JavaScript (jQuery)
- Font Awesome & Bootstrap Icons

## Project Structure

```
IJ/
├── public/              # PHP pages
│   ├── restaurant.php
│   ├── restaurant-about.php
│   ├── restaurant-menu.php
│   └── ...
├── css/                 # Stylesheets
├── js/                  # JavaScript files
├── images/              # Image assets
├── fonts/               # Font files
└── index.php            # Main entry point
```

## Setup

1. Clone the repository
2. Place files in your web server directory (e.g., XAMPP htdocs)
3. Access via `http://localhost/IJ/`
